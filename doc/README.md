## 📖 About VeloxPark

VeloxPark is a modern, real-time parking management system built with:
- **Frontend**: React 19 + Vite
- **Backend**: Firebase Realtime Database
- **Hardware**: ESP32 + ANPR (Automatic Number Plate Recognition)
- **Deployment**: Static hosting (Vercel, Netlify, Firebase Hosting)

### Key Features
✅ Real-time vehicle tracking
✅ Automatic number plate recognition
✅ QR-based UPI payments
✅ Admin dashboard with analytics
✅ Manual entry support
✅ Multi-zone parking management
✅ Dynamic pricing (Car, Bike, Truck, EV)
